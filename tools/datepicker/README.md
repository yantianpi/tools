# daterangepicker
bootstrap下的一个时间控件，不支持秒的选择，支持单时间及时间区间选择 参考 http://www.daterangepicker.com/

## 设置项

* startDate
* endDate
* minDate
* maxDate
* dateLimit
* showDropdowns
* showWeekNumbers
* timePicker
* timePickerIncrement
* timePicker12Hour
* singleDatePicker
* ranges
* opens
* format
* separator
* locale
* ...
